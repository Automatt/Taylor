Taylor




Jorge Izquierdo

@izqui9





Library for building HTTP servers in Swift

Features




  • GET and POST requests
  • Extensible middleware
  • Static file server
  • Body parser

Code



let taylor = Taylor(port: 5000)

taylor.get("/") {

   request, response in

    response.stringBody = "Hello World"
    response.send()

    return nil
}

taylor.startListening(forever: true)

Execute



$ ./run
Server running on port 5000
